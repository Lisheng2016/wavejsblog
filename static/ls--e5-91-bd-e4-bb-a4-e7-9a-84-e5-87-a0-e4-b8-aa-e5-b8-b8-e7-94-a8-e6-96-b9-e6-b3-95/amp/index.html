ls 命令的几个常用方法

by admin

2016-11-10
ls命令是linux下最常用的命令: ### 1.最基础的列出文件命令

ls

2.“echo(回显)”命令与一个通配符(*)参数来雷锤目录中的所有文件

echo *

3.“echo(回显)”命令输出目录

echo */

4.列出一个目录中的所有文件， 包括隐藏的以“.”开头的文件

ls -a

5.列出目录中除了 “当前目录 .”和“父目录 ..”之外的所有文件，包括隐藏文件

ls -A

6.长格式打印出当前目录内容

ls -l

drwxr-xr-x 5 root root 4096 Nov 30 11:31 Binary

上面的drwxr-xr-x 是文件的权限，分别代表了文件所有者，所属组以及“整个世界”。 所有者具有读(r),写(w)以及执行(x)等权限。 该文件所属组具有读(r)和执行(x)但是没有写的权限，整个世界的其他可以访问到该文件的人也具有相同权限。

  • 开头的‘d’意味着这是一个目录
  • 数字’5’表示链接数为5
  • 文件 Binary归属于用户 “root”以及用户组 “root”
  • Nov 30 11:31 表示文件最后一次的访问日期与时间

7.将目录中的内容以长格式列表打印，并且显示出隐藏的“.文件”

ls -la

7.X 不列出“当前目录”和”父目录”

ls -lA

8.找到每个文件的创建者

ls –author -l

9.对用转义字符打印出非显示字符

ls -b

10.指定特定的单位格式来列出文件和目录的大小

同时使用选项“-block-size=scale”和“-l”，但是需要用特定的单位如M，K等来替换‘scale’参数

ls –block-size=M -l

ls –block-size=K -l

11.列出目录中的文件，但是不显示备份文件，即那些文件名以‘~’结尾的文件

ls -B

12.将目录中的所有文件按照名称进行排序，并显示其最后修改时间信息

ls -cl

13.将目录中的文件按照修改时间进行排序，并显示相关联的信息

ls -ltc

14.控制‘ls’命令的输出颜色的有无

ls –color=never //无色

ls –color=auto //自动

ls –color=always //总是有色

15.只需要列出目录本身，而不是目录的内容

ls -d

16.长格式列表命令”ls -l”创建一个别名“ll”，并将其结果输出到一个文件而不是标准输出

ls -la //find.bashrc

nano .bashrc //输入 alias ll = ‘ls -l’

ll > ll.txt // 将 ll结果输出之ll.txt

nano ll.txt // 查看结果
