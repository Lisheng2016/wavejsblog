Windows7承载式网络设定完整教程

准备工作：
确保驱动已安装
确认无线/有线网卡工作正常
==>

netsh wlan show drivers

==>

netsh wlan set hostednetwork mode=allow
只使用mode=disallow可以直接禁用虚拟Wifi网卡开启成功后，网络连接中会多出一个网卡为“Microsoft Virtual WiFi Miniport Adapter”的无线连接2。若没有,请更新无线网卡驱动。

==>

netsh wlan set hostednetwork ssid=wifiname key=wifipass
注：wifiname为想要设置的wifi热点名称,key=填入密码

==>

netsh wlan start hostednetwork
开启承载网络

==>
设置Internet连接共享：
在“网络连接”窗口中，右键单击已连接到Internet的网络连接。使用路由器，则共享本地连接，如果是拨号连接共享的是宽带连接。选择“属性”→“共享”，勾上“允许其他······连接(N)”并选择创建的wifi热点。

提供共享的网卡图标旁会出现“共享的”字样，表示“本地连接/宽带连接”已共享至“无线连接2”。

至此，WiFi基站已组建好，主机设置完毕。笔记本、带WiFi模块的手机等子机搜索到无线网络ssid后，输入密码。

5.设置静态ip。

手机链接目标网络，右键单击无线网络连接2，选择属性，选择Internet协议版本4，设置静态ip。

参考设置：

ip：192.168.3.1
子网掩码：255.255.255.0
默认网关和DNS无需设置

移动端设置静态IP：

192.168.3.x(x不能和你的虚拟网卡一样)。
网关设置为
192.168.3.1
网络掩码：
255.255.255.0

至此，移动端可正常使用该热点。

另附：

netsh wlan show hostednetwork
查看当前承载网络用户数量，连接情况
