CSS布局模型

CSS布局模型
布局模型和盒模型都是CSS最基础、最核心的概念。布局模型在盒模型之上，不同于普通布局模板和布局样式,CSS布局模板是布局模型的外在的表现形式。

主要的元素布局模型有:

  1. 流动模型(FLow)
  2. 浮动模型(Float)
  3. 层模型(Layer)

流动模型2. 未经过人为设置width:npx，在默认情况下，块状元素的宽度都是100%
3. 在流动模型(Float)下，内联元素都是在其包含元素内从左到右水平分布显示的。

层模型：
CSS中定义了Position来支持层模型来表现层布局。

层模型中有三种定位方式：2. position:absolute
3. position:relative
4. position:fixed

设置绝对属性：position:absolute，将对元素从文档流中提取出来，使用right,left,top,bottom相对于其最接近的一个具有定位属性的父包含块进行绝对定位，若不包含类似的包含块，则相对于body元素，即浏览器窗口。

position:absolute
left:20px
top:50px
==>相对于浏览器窗口向右移动了20px，向下移动了50px。

设置相对属性：position:relative，使用r,l,t,b确定元素在正常文档流中等个便宜位置。相对定位完成过程：先按照static(float)创建一个元素并使其浮动，然后相对于之前的位置移动，移动的方向幅度由四向属性控制，偏移前的位置保留不动==>意思就是移动前初始位置这个元素还是占据着，其他元素无法占据

设置固定定位：position:fixed，相对移动的坐标是视图本身(屏幕内的网页窗口)，不会因为滚动而改变位置，与视图一起保持固定，除非浏览器大小发生变化，或者移动浏览器窗口位置。固定位置的元素始终位于浏览器视图的某个位置，不受文档流的影响，与background-attachment:fixed属性功能是一样的。拖动滚动条时fixed元素在视窗内的位置是不变的
右下角广告：

#adver1{position：fixed;<br></br>
right:0;<br></br>
bottom:0;<br></br>
}<br></br>```
  
 Relative与Absolute同时使用：  
 使用四向参数与position:absolute将该元素与具有position:relative的父元素(前辈元素)进行定位。

相对于元素定位的规范：2. 具有参照属性position:relative的元素必须是absolute元素的父辈元素
3. 参照定位的元素必须加入position:relative
4. 需要定位元素加入position:absolute并使用四向参数定位
