centos7系统语言设置

centos7的locale系统语言设置允许我们明确用户界面与运行服务所使用的语言。

系统级别分locale设置村处于/etc/locale.conf，系统启动时被systemd读取。所有用户与服务都将继承该locale配置。当然了，特定的用户与服务可以拥有自己的locale设置。

1.升级软件

yum update

2.查看系统locale状态

localectl status

结果类似于

localectl status

System Locale: LANG=en_US.utf8
VC Keymap: n/a
X11 Layout: n/a

System Locale 即对应/etc/locale.conf 中:

LANG=en_US.utf8

3.列出系统当前所有可用的locale

localectl list-locales

以下列出en开始的locale

4.设置系统locale

localectl set-locale LANG=en_GB.uft8

5.查看设置结果

localectl status

VC Keymap: n/a
X11 Layout: n/a
