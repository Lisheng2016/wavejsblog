LearningLinux-Vim/Vi Command

Linux:Ubuntu16.04

~sudo chown username filename

更改文件归属于哪个用户

~sudo vim filename

创建一个文件
~”press a” to Insert

处于一个文件中时：

:e fileanme

另外打开filename文件

:q!

强制关闭vim并退出

:wq

保存并退出

:w filename

另存为filename文件

Linux:CentOS6.6

touch /etc/shadowsocks.json

创建json文件

vi /etc/shadowsocks.json

打开文件

”press i” to Insert

”press esc” :wq!

URL加入#的意义与作用
