LearningLinux-Navigation in Linux

List command in linux:

ls -l

list and showing permissons of all the files and folders.Including the day they were modified the group/individual ownership of files and folders.

ls -m

do same list action write everything in a block,usually only list names .

find file command:

sudo find -iname filename

filename*(php* everyfile have php in front part of name)
*.conf(find every conf file )

  • represent for everything in name
    search case-insensitive (Aviod uppercase/lower case issue of finding a paticular file in linux)

delete filer/folder command：

file: sudo rm filename

folder: sudo rm folder -R

rename command:

sudo mv file1 file2

move command:

sudo mv /file1 /etc/file1

copy command:(creating a backup file)
sudo cp file file.bak(backuping files)

To list drives attached to computer

sudo fdisk -l

1.To mount a drive

sudo mount drive folder

2.Mount cdrom

sudo mount /dev/cdrom /folder

3.To unmount drive

sudo umount drive

Use /mnt/subfolder for exclusions later
