启用Kali Linux ssh服务

（1）配置SSH参数

vi /etc/ssh/sshd_config

(按i进行编辑)
删去#PasswordAuthentication no的注释#，如果冒号后为NO，则将NO修改为YES

并将PermitRootLogin without-password的值修改为yes

保存，退出vim

以下是完整的sshd_config

（2）启动SSH服务

/etc/init.d/ssh start

或者

service ssh start

查看SSH服务状态是否正常运行

/etc/init.d/ssh status

或者

service ssh status
