签到日志界面显示 Slim Application Error

经教程导入release中的db-160409.sql后签到日志界面显示 Slim Application Error
原因是没有导入签到日志的数据库表：ss_checkin_log.sql

1.首先查看php错误，确定了原因为数据库表的缺失
2.下载ss_checkin_log.sql以及sp_email_verify.sql并执行导入
3.签到记录恢复正常

php-fpm.conf的内容示意:

[global] error_log = log/error.log log_level = notice [www] catch_workers_output = yes

php.ini的内容示意:

; log_errors ; Default Value: Off ; Development Value: On ; Production Value: On log_error = On error_log = "/usr/local/php/var/log/error_log" error_reporting = E_ALL&~E_NOTICE
