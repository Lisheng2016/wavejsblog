盒模型代码简写：

盒模型的边距：外边距(margin)、内边距(padding)、边框(border)设置边距默认是由顺时针排列的
例如：

margin 12px 14px 15px 20px; ==> margin-top:12px; margin-right:14px; margin-bottom:15px; margin-left:20px;

通常有三种简写方法：2. top、right、bottom、left边距相同，都为20px
margin:10px 10px 10px 10px;>margin 10px;
3. top、bottom,left、right边距相同
margin:10px 20px 10px 20px;>margin 10px 20px;
4. left、right边距相同(20px)
可简写为==>margin:10px 20px 40px；

单位和值：

在网页中设置颜色，有字体颜色color，背景颜色background-color，边框颜色border等
主要设置颜色的方法有：
英文命令颜色：
p{color:red;}

RGB颜色

颜色由R,G,B三色的比例配色
p{color:rgb(133,45,200);}
每一项都是0~255之间的数值
也可以用0%-100%的百分数表示
p{color:rgb(20%,30%,40%);}

十六进制颜色

当前普遍使用的方法,其原理依然为RGB设置，但每一项数值由十六进制数值00-ff代替。
p{color:#00ffff;}

长度值：

目前比较常用的是：px(pixel)、em、%(percent)这三种单位都是相对单位：

像素px:pixel

像素实际指的是屏幕上的小点(CSS规范中假设90px=1inch)。正常情况下与显示器的实际像素有关，大多数设计均倾向于使用px 像素值

em

指的是本元素给字体定义的font-size值，font-size:14px;==>1em=14px
p{font-size:12px;text-indent:2em}
该段代码显示：p元素首行缩进两个字符，也就是24px
