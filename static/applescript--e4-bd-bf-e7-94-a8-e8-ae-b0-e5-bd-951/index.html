最近在使用Applescript，觉得是挺好用的，但是因为和其他脚本语言有些不同，所以在此记录一些遇到的问题和解决方法。

1.关于application “System Preference Pane”

使用脚本操作系统设置的菜单，例如进入子菜单之类的行为，需要知道具体进入的菜单的名称，那么该名称对应的文件应该存在于：

/System/Library/PreferncePanes

如下表给出了所有子菜单的名称

进入子菜单的代码类似于：

tell application "System Preferences" activate set current pane to pane id "com.apple.preference.sound" end tell
