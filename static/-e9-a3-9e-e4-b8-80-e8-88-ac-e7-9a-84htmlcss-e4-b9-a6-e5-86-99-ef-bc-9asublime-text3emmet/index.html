使用Emmet将只需要输入一行语句即可完成上述的所有标签创建

在前端开发的过程中，一个最繁琐的工作就是写 HTML、CSS 代码。数量繁多的标签、属性、尖括号、标签闭合等，让前端们甚是苦恼。而Emmet，它提供了一套非常简单的语法规则，书写起来非常爽快，然后只需要敲一个快捷键就立刻生成对应的 HTML 或 CSS 代码，极大提高了代码书写效率。

page>div.logo+ul#navigation>li>a{item $*5}/继续输入tab/

生成 HTML 文档初始结构

HTML的结构性信息是比较固定的,类似!DOCTYPE HTML、meta、title等，EMMET可利用!或者html:5轻松创建Html5标准的标准结构。

如果需要生成Html4过度型结构,输入html:xt即可生成：

同理，可利用html:4s 生成 HTML4 严格型

总结：- html:5或！生成HTML5结构

  • html:xt生成HTML4过渡型结构
  • html:4s生成HTML4严格型结构

HTML书写的基本语法：- X代表HTML标签

  • X#id表示X标签id属性
  • X.class表示X标签class类
  • X[attr=example]表示X标签attr=example属性
  • X{a}表示X标签的a内容
  • X>B表示X标签与下级标签B
  • X+B表示X、B为同级标签
  • X^N表示N是X的上级元素

CSS书写

发现在html文本编辑中，并不能直接使用emmet简写完成css的简写，可能需要在css文件，才能使用简写，主要与EMmet判断的当前的文本类型有关。

主要体现在简写：

posa == position:absolute w100 == width:100px

简写单位别名：- p表示%

  • e表示em
  • x表示ex

简写EMMET代码:(Emmet指令中不允许存在空格)

m10-20 生成margin:10px 20px;(简写单位一致则使用横杠分离两个不同的值)

如需生成负值，则为

m10–20(多加一条横杠)

/如已经为每个值指定单位，那么不需要输入分隔的横杠/ m10e20e生成: margin:10em 20em; /同样情况使用横杠分隔可以生成负值/ m10e-20e生成: margin:10em -20em; /同时生成多的值：(使用+连接两者)/ h10p+m5e生成: height:10%;margin:5em;

颜色值的生成：

c#3生成color:#333
bd5#0s:border:5px #000 solid;

颜色值生成的规则为：2. #1=#111111
3. #e0=#e0e0e0
4. #fc0=#ffcc00
